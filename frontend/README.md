# CapMint — Frontend Portal (@capmint/frontend)

## 1. Overview
The `@capmint/frontend` package contains the high-fidelity, interactive user interface for the CapMint platform.

It is implemented as a single-page simulation app containing:
1.  **Certifier Portal**: Key authorization checks, budget approvals, and cascade batch invalidation controls.
2.  **Manufacturer Console**: Remaining budget gauges and secure random QR code serial generators.
3.  **Consumer Scanner PWA**: Mobile phone interface allowing scanning codes and testing geovelocity clone detection presets.
4.  **Transparency Ledger**: Verification tree displaying blockchain log chain blocks.

---

## 2. Design Aesthetics
*   **Typography**: Outfits (headings) and Inter (body).
*   **Theme**: Sleek Dark Glassmorphism, styled with CSS variables and glowing neon accents.
*   **Aesthetics**: Fully responsive layout structure with micro-animations.
