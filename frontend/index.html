Geovelocity Clone Detection Scenario

To test the clone detection engine, try executing the following sequence:

  1. Mint QR codes in the Manufacturer panel.
  2. Go to the Mobile Scanner, set the location preset to London, and click a QR code. Notice it reports VERIFIED.
  3. Change the location preset to Tokyo, and scan the same QR code again.
  4. The system will calculate that the physical object traveled from London to Tokyo in 0.0 seconds (exceeding 800 km/h), flag the code as CLONE-SUSPECT, and alert the consumer.
