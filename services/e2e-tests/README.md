# CapMint — End-to-End Integration Tests (@capmint/e2e-tests)

## 1. Overview
This package implements the end-to-end integration and lifecycle verification tests for the CapMint platform.

It consolidates validations for all 11 backend core modules (`M-001` through `M-011`) in a single sequential flow.

---

## 2. Test Execution
To run E2E integration tests:
```bash
npm run test --workspace=@capmint/e2e-tests
```
